Logger

Module for working with events You can get the value returned by the scheduled task, and you can also get the errors that occur during the execution of the task when using events. More about how events work: https://nodejs.org/api/events.html

# Error

Allows you to handle events with an error structure (ErrorLog)

View Source logger.ts, line 24

Example
// Handling errors
const scheduler = require(`simple-scheduler-task`);
scheduler.events.on("errors", function (error) {
   console.log(error); // => Error structure
});
